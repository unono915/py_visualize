# py_visualize

브라우저에서 바로 실행되는 수업용 Python 변수 변화 시각화 도구입니다.  
Pyodide를 사용해 별도 서버 백엔드 없이 Python 코드를 실행하고, 각 줄이 실행될 때 변수 생성·변경·사용 흐름을 장면처럼 보여줍니다.

## 핵심 기능

- Python 코드를 브라우저에서 직접 실행
- `한 줄 실행`과 단계별 탐색
- 변수 노드 시각화와 연결선 표시
- 생성/변경 변수 애니메이션
- `print()` 출력 콘솔
- `input()` 입력 모달 지원
- 리스트/딕셔너리 미리보기와 상세 보기
- 변수 노드 드래그 이동

## 지원 범위

이 프로젝트는 범용 디버거가 아니라 초급 Python 수업용 도구입니다. 현재는 아래 범위를 중심으로 설계되어 있습니다.

- 변수 대입
- 문자열, 숫자, bool, None
- 리스트 / 튜플 / 셋 / 딕셔너리 기초
- `print()`
- `input()`
- `for` 반복문
- 기초 함수 호출

## 실행 방법

빌드 도구는 필요하지 않습니다.

### 1. 파일을 바로 열기

`index.html`을 브라우저에서 직접 열어도 실행할 수 있습니다.

### 2. 로컬 서버로 실행하기

```bash
python -m http.server 8000
```

브라우저에서 아래 주소로 접속합니다.

```text
http://127.0.0.1:8000/
```

## 사용 방법

1. 왼쪽 코드 에디터에 Python 코드를 입력합니다.
2. `실행`을 누르면 전체 코드를 실행하고 최종 결과를 보여줍니다.
3. `한 줄 실행`을 누르면 첫 단계부터 시작하며, 오른쪽 `다음` 버튼으로 흐름을 따라갈 수 있습니다.
4. 오른쪽 시각화 영역에서 변수 노드와 연결선을 보며 어떤 줄에서 어떤 변수가 사용되는지 확인합니다.
5. 잘린 리스트/딕셔너리 노드는 마우스를 올리면 상세 내용을 볼 수 있습니다.

## 주의 사항

- 첫 실행 시 Pyodide 로딩 때문에 수 초에서 수십 초가 걸릴 수 있습니다.
- 무한 반복문은 보호 장치로 중단되지만, 아주 복잡한 코드는 이 도구의 목표 범위를 벗어납니다.
- 학습용 시각화를 우선하므로 모든 Python 문법을 완전하게 다루지는 않습니다.
